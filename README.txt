Привет, парсер срабатывает каждый час в 10 минут. Результат записывает в файл json, находящийся по пути VSTU-timetable-TelegramBot/parsertimetable.json
Достать расписание из файла можно, используя следующий код: import json;  with open('', 'r') as file: passes = json.load(file);. 
Также необходимо выполнить настройу: заменяешь значение переменной programm_path в main.py на путь к папке, в которой находиться папка проекта.
